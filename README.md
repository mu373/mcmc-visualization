# mcmc-visualization

Interactive 3D visualization of Markov Chain Monte Carlo sampling algorithms.  
https://mcmc-visualization.vercel.app/

## Algorithms

- Random Walk Metropolis-Hastings
- Hamiltonian Monte Carlo (HMC)
- No-U-Turn Sampler (NUTS)
- Metropolis-adjusted Langevin (MALA)
- Gibbs Sampler

## Distributions

- Gaussian
- Quartic Gaussian
- Bimodal
- Multimodal
- Banana
- Donut
- Squiggle
- Rosenbrock
- Rastrigin
- Ackley

## Keyboard shortcuts
- Space: Play/Pause
- N: Step

## Development

```bash
pnpm install
pnpm dev
```
